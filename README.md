# Projeto Tabuada

Esse projeto é sobre uma tabuada de multiplicação. Pode multiplicar quantas vezes quiser e, por vários números diferentes.

## 🔥 Introdução

Para treinar o JavaScript, fiz esse projetinho de uma Tabuada, usei o HTML e CSS também.
